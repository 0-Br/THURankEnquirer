# THURankEnquirer

清华大学学生成绩排名批量查询与统计分析工具。

通过[问卷星](https://www.wjx.cn)成绩查询接口，批量获取学生的年级与班级排名百分位数据，按自定义评分规则计算综合得分并分档导出。

## 工作流程

本项目包含三个脚本，依次执行完成从数据提取到结果导出的完整流程。

### 第一步：`extraction.py` — 数据预处理

从原始学生花名册 Excel 中提取学号、姓名与身份证号，按学号范围筛选目标年级学生，输出为 CSV 格式供后续使用。

### 第二步：`crawler.py` — 排名数据爬取

逐条读取 CSV 中的身份证号，向问卷星成绩查询页面发送 POST 请求，解析返回 HTML 中的四项排名百分位数据：

| 排名维度 | 说明 |
|---------|------|
| 年级排名（全部） | 全部课程的年级排名百分位 |
| 年级排名（必限） | 必修与限选课程的年级排名百分位 |
| 班级排名（全部） | 全部课程的班级排名百分位 |
| 班级排名（必限） | 必修与限选课程的班级排名百分位 |

爬取过程支持随机 User-Agent、失败自动重试（最多 30 次）与随机延时（0–5 秒），结果以 pickle 格式保存。

### 第三步：`results.py` — 评分与分档导出

加载爬取结果，对每项排名百分位按以下规则评分：

| 排名百分位 | 得分 |
|-----------|------|
| 前 30% | 25 分 |
| 30%–50% | 15 分 |
| 50%–80% | 10 分 |
| 80%–100% | 5 分 |

四项排名得分相加，得到 0–100 分的综合评分，再按分数段分为 A–F 六档：

| 等级 | 分数区间 |
|------|---------|
| A | ≥ 90 |
| B | 75–89 |
| C | 60–74 |
| D | 45–59 |
| E | 30–44 |
| F | < 30 |

最终结果导出为包含 7 个 sheet 的 Excel 文件（全体 + A–F 各档）。

## 依赖

- Python 3
- pandas, openpyxl
- requests, beautifulsoup4
- fake-useragent, tqdm

## 注意事项

`data/` 目录包含学生个人信息（学号、姓名、身份证号、排名数据），已通过 `.gitignore` 排除，不纳入版本控制。
